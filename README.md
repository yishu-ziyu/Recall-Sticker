# Recall Sticker 📝

> **"高亮即挖空"** —— 一个低摩擦的网页记忆工具。

将网页阅读变成主动回忆。选中文字，一键遮盖，原地复习。利用"场景记忆"，在文字所在的上下文中强化记忆。

## ✨ 核心功能

| 功能              | 描述                                     |
| ----------------- | ---------------------------------------- |
| 🖍️ **划词贴纸**   | 选中任意文字，一键将其"挖空"变成灰色贴纸 |
| ⌨️ **键盘导航**   | `Tab` 跳转，`Space` 揭示，零鼠标复习     |
| 👁️ **偷看模式**   | 按住 `Alt` 键快速预览所有答案            |
| 🔄 **会话重置**   | 一键重新隐藏所有贴纸，开始新一轮复习     |
| 📥 **Anki 导出**  | 智能截取完整句子，自动生成 Cloze 卡片    |
| 🗂️ **侧边栏回顾** | 集中展示所有贴纸，点击可精准定位回原文   |

## 🚀 安装

1.  下载或克隆本仓库
2.  打开 Chrome，访问 `chrome://extensions`
3.  开启右上角 **"开发者模式"**
4.  点击 **"加载已解压的扩展程序"**
5.  选择 `Recall-Sticker` 文件夹

## 📖 使用方法

1.  **创建贴纸**: 在任意网页上选中文字 → 点击浮现的 🖍️ 按钮（或按 `Alt + S`）
2.  **揭示/隐藏**: 点击灰色贴纸，或用键盘 `Tab` + `Space`
3.  **偷看全部**: 按住 `Alt` 键
4.  **删除贴纸**: 右键点击贴纸 → 确认删除
5.  **侧边栏回顾**: 点击扩展图标 → "Open Side Panel" 查看所有记录
6.  **导出到 Anki**: 点击右下角控制台的 📥 按钮

## 💾 数据存储

所有贴纸数据通过 Chrome Storage API 存储在本地，按 URL 路径（不含参数）分组。这意味着：

- 你的数据完全在本地，不会上传到任何服务器。
- 刷新页面后贴纸自动恢复。
- 同一页面的不同 URL 参数（如 `?utm_source=...`）共享贴纸。

## 🎨 设计理念

- **低摩擦 (Low Friction)**: 不打断阅读心流，高亮即完成。
- **原位复习 (In-Situ Review)**: 利用场景记忆，在原网页上复习。
- **零后台 (Zero Backend)**: 纯本地运行，无需账号，无需联网。
